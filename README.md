# Interactive

Проект **Interactive** на React + TypeScript + Webpack.  
Отображает круг с точками, слайдер событий, динамическую навигацию.

## Установка

Клонируем репозиторий:

```bash
git clone https://github.com/StasZ2023/Interactive.git
cd interactive

npm install
# или
yarn install

npm run dev
# или
yarn run dev
